# GitMind Self-hosted

## Opis produktu (do Gumroad)

### Krótki opis:
**AI Code Review Assistant for GitLab - zainstaluj na swoim serwerze**

GitMind automatycznie analizuje Merge Requesty i wykrywa bugi, problemy bezpieczeństwa i błędy jakości. Wszystko na Twoim serwerze - pełna kontrola, zero external dependencies.

### Pełny opis:

**Co dostajesz:**
- Pełny kod źródłowy GitMind (Python/Flask)
- Docker configuration dla łatwego wdrożenia
- Instrukcja instalacji krok po kroku
- Konfiguracja webhook GitLab
- 3 miesiące darmowych aktualizacji

**Funkcje:**
- Automatic code review dla Merge Requestów
- Wykrywanie hardcoded secrets i API keys
- Feedback loop - agent uczy się z Twoich akceptacji
- Emoji i statystyki w komentarzach
- Rate limiting i input validation
- Monitoring i alerting

**Wymagania:**
- Python 3.11+
- Docker (opcjonalnie)
- Konto GitLab z Personal Access Token
- Konto Groq (darmowe, api.groq.com)

**Wsparcie:**
- Dokumentacja online
- 3 miesiące aktualizacji w cenie
- Email support
