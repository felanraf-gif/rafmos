Dostajesz link do pobrania kodu źródłowego GitMind (ZIP), dokumentację instalacji, przykładową konfigurację i license key ważny przez 1 rok.

Czy mogę używać na wielu projektach?

License key jest ważny dla 1 projektu GitLab. Potrzebujesz więcej? Napisz do mnie po indywidualną ofertę.

Licencja ważna 1 rok od zakupu. Odnowienie kosztuje $9/rok i zawiera dostęp do wszystkich aktualizacji.

Python 3.11+, Docker (opcjonalnie), konto GitLab z Personal Access Token, konto Groq API (darmowe na api.groq.com).
